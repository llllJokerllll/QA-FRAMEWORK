# QA-Framework Documentation

Welcome to the QA-Framework documentation!

## Overview

QA-Framework is a modern QA automation framework built with Clean Architecture and SOLID principles.

## Features

- Multi-framework testing (Selenium, Playwright, Appium, Cypress)
- Dashboard web with React + FastAPI
- Integration Hub (Jira, Zephyr, Azure DevOps, TestLink)
- Comprehensive testing suite
- CI/CD automation
